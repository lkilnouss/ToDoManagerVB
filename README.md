A simple CLI **Task Manager** as a learning project in .NET 
##
`Features`: 
- add tasks
- delete tasks
- view tasks
- mark tasks as complete
- create *.ics* files ready to import to your calendar
##


`Usage`:
- "help"               : display all available commands
- "add"                : add new task
- "list"               : list all tasks (present and previously completed)
- "list open"          : list only open tasks
- "list done"          : list only done tasks
- "delete"             : deletes a task
- "delete all"         : deletes all stored tasks
- "complete"           : completes a task
- "create appointment" : creates a .ics calendar file 

-> use specified commands. You will be asked for the project ID for completing and deleting tasks, which you can find via "list"
